# MolSysSuite Papers

This repository contains scientific papers associated with the **MolSysSuite** ecosystem.

Each paper lives in a tool-specific subdirectory (e.g. `molsysmt/`, `molsysviewer/`, `molsys-ai/`)
and is self-contained, with its own README, manuscript sources, figures, and optional artifacts.

This repository is the **single source of truth for papers**. Individual tool repositories
only contain metadata pointers in `resources/papers.yml`.

## Structure

- `<tool>/<year>-<slug>/`
  - `README.md` – paper description and status
  - `manuscript/` – LaTeX/Markdown sources
  - `figures/`
  - `bibliography/`
  - `artifacts/` – PDFs / supplementary (optional)

## Status conventions

- `idea`
- `in_preparation`
- `submitted`
- `accepted`
- `published`
- `archived`
